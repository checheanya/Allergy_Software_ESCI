@startuml
actor User
participant "Allergy App" as App
participant "User alert data" as da

User -> App: Open app
App -> App: Show Home Screen
User -> App: Select "Receive Alert from Database"
App -> App: Retrieve Nearby User Alert
App -> da: Request Nearby User Alert
da -> App: Send Nearby User Alert
App --> User: Display Nearby User Alert
@enduml
