@startuml

actor Health Tracking App
participant "Alert System" as System
database "Patient Database" as Database

Health Tracking App -> System: Update Daily Metrics
System -> Database: Record Metrics to Database
System -> Database: Update Patient's Dynamic Data
System -> System: Check Metrics
alt Metrics are High
  System --> Health Tracking App: Send Notification with Quiz Suggestion
else Metrics are not High
  System -> System: No Action
end

@enduml
